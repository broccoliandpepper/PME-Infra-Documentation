# 07 - Licensing & Services

## Overview

This folder documents the licensing model and cloud services used by K Creation & Production across the organisation.

All users are assigned a consistent, standardised licensing package that provides both productivity tools and security services.

## Contents

- `m365-business-premium.md` – Microsoft 365 Business Premium licence.
- `defender-endpoint-p2.md` – Microsoft Defender for Endpoint P2 add-on.

## Licensing Model

| Licence | Assigned To | Purpose |
|---------|-------------|---------|
| **Microsoft 365 Business Premium** | All users | Productivity and collaboration tools |
| **Microsoft Defender for Endpoint P2** | All users (add-on) | Advanced endpoint security and threat protection |

## Scope

The licensing applies to:

- All employees across departments (Accounting, HR, Production, etc.).
- All user workstations and devices (Windows, Mac).
- All cloud and on-prem users.

## Key Benefits

- **Standardisation**: Consistent set of tools across the organisation.
- **Security**: Unified threat detection and response capabilities.
- **Productivity**: Access to essential Microsoft cloud services.
- **Management**: Centralised licensing and deployment via Intune.

## Integration with Other Components

These licences are the foundation for:

- **Intune**: Device management and app deployment.
- **Entra ID**: User identity and conditional access.
- **Microsoft Sentinel**: Security monitoring and incident response (if enabled).
- **Azure Defender**: Cloud workload protection (if used).

Each licencing choice directly supports other infrastructure components documented in the main repository.
